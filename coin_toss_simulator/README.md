How to Play
Start: The game initializes by asking if you are ready to play and requesting your name.
Gameplay Loop:
User's Turn: You predict the coin flip by entering "H" (Head) or "T" (Tail). The game randomly flips a coin and checks if your guess matches.
Computer's Turn: The computer makes a random guess ("H" or "T"), and the game checks if it matches the coin flip.
Feedback: Immediate feedback is given for both turns (e.g., "Correct_answer!", "Oops!", etc.).
Scoring & Levels:
The game tracks scores for both the user and the computer.
A "Level" consists of 10 rounds.
At the end of a level, a Score Board is displayed showing the winner of that level.
If there is a winner, a graphical window pops up (using turtle) to announce the winner's name in large text.
Continuation: After a level ends, you can choose to play the next level (resetting scores but keeping the session alive) or exit.
